# NEXORA — Web App Studio

ต้นแบบเว็บไซต์ภาษาไทยสำหรับ Digital Product Studio ที่รับออกแบบและพัฒนา Web Application แบบครบวงจร

## วิธีเปิดดู

เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ได้ทันที โดยไม่ต้องติดตั้งโปรแกรมเพิ่มเติม

## สิ่งที่มีในต้นแบบ

- Responsive layout สำหรับ Desktop, Tablet และ Mobile
- เมนูมือถือและ Smooth scroll
- Motion Background แบบ Aurora, Data signal และแสงเคลื่อนไหว
- บริการ ประเภทระบบ กรณีศึกษา ขั้นตอนทำงาน และสิ่งส่งมอบ
- FAQ แบบเปิด–ปิด และ Project brief ที่เก็บข้อมูลสำคัญครบ
- Scroll reveal, Mouse parallax และ micro-interactions
- ฟอนต์ Noto Sans Thai และ Space Grotesk ในโทนทันสมัยเป็นทางการ

ข้อมูลชื่อแบรนด์ อีเมล ผลงาน และปีในหน้าเว็บเป็นข้อมูลตัวอย่าง สามารถแก้ได้จาก `index.html`
